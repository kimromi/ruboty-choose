# Ruboty::Choose

Choose tool for Ruboty.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruboty-choose'
```

## Usage

Select one of multiple parameters.

```
@ruboty choose A B C   => I chose B!
```
